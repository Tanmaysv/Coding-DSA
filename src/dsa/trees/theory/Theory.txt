Trees are non linear data structure

Root:Topmost-Element of tree is Root - Root node dont have parent

leaf:These are node who dont have child

Siblings: children of same parent

edge:Link between parent and child

Height: Height can be of a node or of a tree

1)Height of node: height from that node to bottom node

2)Height of tree: Height of that tree from root to bottom


Level/Depth: level can be of a node or of a tree

1)Level of node: level from root node to that node

2)level of tree: level of that tree from root to bottom

Height and level/depth of tree are always equal
